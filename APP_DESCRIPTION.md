## 一、产品概述

FLOB2B 是一个面向专业交易者和跨境电商卖家的 **AI驱动自动化平台**，核心功能包括 **Polymarket智能交易机器人** 和 **云浏览器指纹环境管理系统**。平台通过浏览器指纹隔离技术、住宅代理IP池和AI策略生成，为用户提供安全、稳定、自动化的交易和账号运营能力。

## 二、核心功能模块

### 2.1 Polymarket 智能交易机器人 (Poly Bot)

#### 功能定位
旗舰级预测市场自动化交易模块，支持模拟盘测试和实盘自动交易。

#### 核心能力
| 功能 | 描述 |
|------|------|
| **市场发现** | 多线程机器人实时爬取Polymarket市场数据，监控流动性、价格波动 |
| **AI策略生成** | 用户用自然语言描述策略，AI自动生成执行代码（如"跟踪鲸鱼0x123...，当其买入>1000 USDC时跟随买入YES"） |
| **模拟交易 (Paper)** | 虚拟资金测试策略，支持并发度10，本地数据库存储交易记录 |
| **实盘交易 (Live)** | 服务器端执行，并发度限制为1防止账户风控，加密私钥链上自动执行 |
| **风险管理** | 日预算控制、单笔上限、自动止损、仓位管理 |
| **实时监控** | PnL实时更新、交易历史、策略表现分析 |

#### 技术特点
- **钱包认证**: 连接钱包授权 + 加密私钥服务器端签名
- **IP隔离**: 每个用户绑定专属住宅IP，防止关联
- **执行约束**: 实盘交易强制单并发，模拟盘支持高并发测试

### 2.2 云浏览器指纹环境 (Cloud Browser)

#### 功能定位
为跨境电商、社交媒体运营提供防关联的多账号管理环境。

#### 核心能力
| 功能 | 描述 |
|------|------|
| **指纹环境生成** | 自动生成唯一浏览器指纹（Canvas/WebGL/AudioContext噪音注入、User-Agent、时区、字体等） |
| **IP绑定** | 支持平台共享IP（24小时轮换）或购买专属住宅IP（$40/年，固定不变） |
| **账号集管理** | 按业务创建环境组（如"TikTok美区店群"、"FB广告账号"） |
| **一键启动** | Web端直接启动隔离浏览器，无需本地安装 |
| **会话持久化** | Cookies、登录态、LocalStorage自动保存，下次启动秒恢复 |
| **域名白名单** | 环境仅限访问指定域名（如*.tiktok.com, *.facebook.com），防止数据泄露 |
| **WebRTC防护** | 禁用或代理WebRTC，防止真实IP泄露 |

#### 适用场景
- TikTok Shop多店铺运营
- Facebook/Google广告账号管理  
- Amazon多账号防关联
- 任何需要多账号隔离的Web平台

### 2.3 代理与IP管理系统

#### 功能定位
全局代理资源池管理，为交易机器人和云浏览器提供网络隔离。

#### 核心能力
| 功能 | 描述 |
|------|------|
| **住宅IP池** | $40/年/个的专属住宅IP，静态固定，模拟真实用户 |
| **IP分配** | 管理员后台分配IP给用户，用户可查看自己的专属IP |
| **DNS过滤** | CoreDNS拦截非白名单域名，防止环境被滥用 |
| **流量监控** | 每个IP的流量统计和异常告警 |
| **黑名单防护** | 阻止跟踪像素、分析脚本等无关请求 |

### 2.4 SaaS 运营管理系统

#### 2.4.1 用户与计费
| 功能 | 描述 |
|------|------|
| **多租户体系** | 支持普通用户和管理员两种角色 |
| **订阅管理** | 交易节点月费$49/月，IP资源$40/年 |
| **积分系统** | Credits用于API调用、策略执行等资源消耗 |
| **账单历史** | 完整支付记录和发票管理 |
| **分层套餐** | Basic/Professional/Enterprise 三档服务 |

#### 2.4.2 管理员后台
| 功能 | 描述 |
|------|------|
| **用户管理** | 查看所有用户、封禁/解封、额度调整 |
| **财务看板** | 收入统计、付费转化率、退款管理 |
| **系统监控** | 服务器CPU/内存、活跃Bot进程、IP池健康度 |
| **环境审计** | 查看全平台活跃的指纹环境列表 |
| **IP库存** | 住宅IP的采购、分配、回收管理 |

## 三、技术架构

### 3.1 前端技术栈
| 技术 | 用途 |
|------|------|
| React + TypeScript | 核心框架 |

| 技术 | 用途 |
|------|------|
| 指纹噪音注入 | Canvas/WebGL/AudioContext 防检测 |

## 四、用户使用流程

### 4.1 交易用户旅程
```
1. 注册 → 2. 订阅节点($49/月) → 3. 绑定钱包 → 4. 配置策略
   → 5. 模拟测试 → 6. 启动实盘 → 7. 监控PnL
```

### 4.2 电商用户旅程
```
1. 注册 → 2. 购买住宅IP($40/年) → 3. 创建环境组
   → 4. 生成指纹环境 → 5. 一键启动云浏览器
   → 6. 登录TikTok/Amazon → 7. 日常运营（会话自动保持）
```
